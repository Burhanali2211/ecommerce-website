## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Burhanali2211/ecommerce-website.git
   ```

2. Install dependencies:
   ```bash
   cd ecommerce-website
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
